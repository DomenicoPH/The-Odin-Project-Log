Preguntas

¿Qué es un servidor web?

Un software o computadora que almacena y entrega páginas web a los navegadores.

¿Qué es una network (red)?

Un conjunto de dispositivos conectados que pueden intercambiar información.

¿Qué es Internet?

Una red mundial de redes conectadas entre sí.

¿Qué es una dirección IP?

Un identificador único que permite localizar un dispositivo en una red.

¿Qué es un router?

Un dispositivo que dirige el tráfico de datos entre redes.

¿Qué es un ISP?

Un proveedor de servicios de Internet (por ejemplo, Movistar o Claro).

¿Qué son los packets y cómo se usan para transferir datos?

Son pequeños bloques de información. Los datos se dividen en paquetes, se envían por la red y se reconstruyen al llegar al destino.

¿Qué es un cliente?

Un programa o dispositivo que solicita servicios o datos a un servidor.

¿Qué es un servidor?

Un programa o dispositivo que proporciona servicios o datos a los clientes.

¿Qué es una página web?

Un documento accesible mediante un navegador.

¿Qué es un buscador web?

Un sitio web que permite buscar información en Internet (ej.: Google).

¿Qué es un motor de búsqueda?

El sistema que indexa y encuentra páginas web para responder búsquedas.

¿Qué es una petición DNS?

Una consulta para convertir un nombre de dominio (ej. google.com) en una dirección IP.
