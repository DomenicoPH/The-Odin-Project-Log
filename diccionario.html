Diccionario

L

  • LTS:

    (Long Term Support) Versión de un programa, sistema operativo o plataforma de desarrollo que recibe actualizaciones de seguridad y mantenimiento técnico garantizadas durante un período prolongado (generalmente de 2 a 5 años o más), priorizando la estabilidad del sistema sobre la introducción de nuevas funciones.

S

  • SSH:

    (Secure Shell) Protocolo de red seguro que permite a los usuarios acceder, administrar y transferir datos de forma remota en una computadora o servidor a través de un canal de comunicación completamente cifrado.

    wikipedia.org

  • SSR:

    (Server Side Rendering) Técnica de desarrollo web en la que el servidor procesa el código de una aplicación para generar la página HTML completa ya armada y enviarla directamente al navegador del usuario, mejorando la velocidad de carga inicial y el posicionamiento en buscadores (SEO).

    developer.mozilla.org

U

  • URL:

    (Uniform Resource Locator) Dirección única y específica que se asigna a cada uno de los recursos disponibles en la red (como una página web, una imagen o un archivo) para que puedan ser localizados y accedidos a través de un navegador de internet.

    wikipedia.org
